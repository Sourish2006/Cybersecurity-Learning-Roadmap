# Password Security

## 📌 Overview

Password security refers to the practices and techniques used to protect user accounts and sensitive information from unauthorized access.

Strong passwords are one of the most important components of cybersecurity and serve as the first line of defense against attackers.

---

## 🎯 Why Password Security is Important

Password security helps to:

* Prevent unauthorized access
* Protect sensitive information
* Reduce the risk of account compromise
* Enhance overall cybersecurity

Weak passwords are one of the leading causes of security breaches.

---

## 🔹 Characteristics of a Strong Password

A strong password should include:

* Uppercase letters (A-Z)
* Lowercase letters (a-z)
* Numbers (0-9)
* Special characters (!@#$%^&*)
* Sufficient length (12+ characters)

Example:

```text
S3cur3@Pass2026
```

---

## 🔹 Characteristics of a Weak Password

Weak passwords often:

* Are too short
* Contain common words
* Use predictable patterns
* Lack complexity

Examples:

```text
password
123456
qwerty
admin
```

---

## 🔹 Common Password Threats

Attackers use various techniques such as:

* Brute-force attacks
* Dictionary attacks
* Credential stuffing
* Password spraying

---

## 🌍 Real-World Applications

Password security is important for:

* Online accounts
* Banking systems
* Email services
* Enterprise networks
* Authentication systems

---

## ⚠️ Security Best Practices

* Use long passwords
* Avoid common passwords
* Use unique passwords for each account
* Enable Multi-Factor Authentication (MFA)
* Use a password manager
