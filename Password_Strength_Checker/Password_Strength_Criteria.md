# Password Strength Criteria

## 📌 Overview

Password strength criteria are a set of rules used to evaluate how secure a password is against common attacks.

A strong password reduces the likelihood of unauthorized access and improves overall account security.

---

## 🎯 Why Password Strength Matters

Strong passwords help to:

* Protect user accounts
* Prevent unauthorized access
* Resist brute force attacks
* Reduce the risk of password guessing
* Improve cybersecurity posture

---

## 🔹 Length

Password length is one of the most important factors in determining password strength.

### Recommended Length

* Minimum: 8 characters
* Strong: 12+ characters
* Highly Secure: 16+ characters

Example:

```text id="2sd9kz"
S3cur3@Password2026
```

Longer passwords are generally more difficult to crack.

---

## 🔹 Uppercase Letters

Including uppercase letters increases password complexity.

Example:

```text id="yx10wl"
A-Z
```

---

## 🔹 Lowercase Letters

Lowercase characters add additional combinations.

Example:

```text id="s2grc0"
a-z
```

---

## 🔹 Numbers

Numbers improve password diversity.

Example:

```text id="0z4qq2"
0-9
```

---

## 🔹 Special Characters

Special characters make passwords harder to guess.

Examples:

```text id="04vm73"
! @ # $ % ^ & *
```

---

## 🔹 Common Password Detection

Passwords such as:

```text id="g5mvya"
password
123456
admin
qwerty
password123
```

are considered weak because they are frequently targeted in dictionary attacks.

---

## 🔹 Repeated Characters

Repeated patterns reduce password strength.

Examples:

```text id="j0cgqb"
aaaaaa
111111
```

---

## 🔹 Sequential Patterns

Predictable sequences should be avoided.

Examples:

```text id="j6yvkl"
123456
abcdef
qwerty
```

---

## 🔹 Password Strength Levels

| Score | Strength    |
| ----- | ----------- |
| 0–2   | Weak        |
| 3–4   | Moderate    |
| 5–6   | Strong      |
| 7+    | Very Strong |

---

## 🌍 Real-World Applications

Password strength criteria are used in:

* Authentication systems
* Registration forms
* Password managers
* Enterprise security solutions

---

## ⚠️ Security Best Practices

* Use unique passwords
* Avoid password reuse
* Use long passwords
* Enable Multi-Factor Authentication (MFA)
* Use a password manager
