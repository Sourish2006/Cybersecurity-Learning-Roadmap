# Encryption Basics

## 📌 Overview

Encryption is the process of converting readable data (**plaintext**) into an unreadable format (**ciphertext**) to protect it from unauthorized access.

Only users possessing the correct key can decrypt the ciphertext back into plaintext.

Encryption is one of the fundamental pillars of cybersecurity and plays a critical role in protecting data confidentiality.

---

## 🎯 Objectives of Encryption

* Ensure confidentiality of sensitive information
* Prevent unauthorized access
* Protect data during storage and transmission
* Enhance privacy and security

---

## 🔹 Basic Terminology

| Term       | Description                               |
| ---------- | ----------------------------------------- |
| Plaintext  | Original readable data                    |
| Ciphertext | Encrypted unreadable data                 |
| Key        | Value used for encryption and decryption  |
| Encryption | Converting plaintext into ciphertext      |
| Decryption | Converting ciphertext back into plaintext |

---

## 🔹 Types of Encryption

### 1. Symmetric Encryption

* Uses the same key for encryption and decryption
* Faster and simpler
* Suitable for large amounts of data

**Examples:**

* AES
* DES

---

### 2. Asymmetric Encryption

* Uses a pair of keys (Public Key and Private Key)
* More secure for communication

**Examples:**

* RSA
* ECC

---

## 🔹 Encryption Process

```text
Plaintext
     ↓
Encryption Algorithm + Key
     ↓
Ciphertext
     ↓
Decryption Algorithm + Key
     ↓
Plaintext
```
